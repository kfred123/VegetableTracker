# 🥦 VegetableTracker

A cross-platform mobile app built with **.NET MAUI** that helps you track your vegetable intake and reach the recommended goal of **30 different vegetables per week**.

## 🎯 Goal

Nutritional guidelines suggest eating **30 different types of vegetables per week** for optimal gut health and nutrition. VegetableTracker makes it easy to monitor your progress and discover new vegetables to add to your diet.

## ✨ Features

- **Log Vegetables** — Quickly log the vegetables you've eaten each day.
- **7-Day Progress** — See your completion percentage based on the last 7 days (rolling window).
- **Suggestions** — Get personalized suggestions for vegetables you haven't eaten recently to help you reach 100%.
- **Cross-Platform** — Runs on Android, iOS, Windows, and macOS thanks to .NET MAUI.

## 📊 How It Works

1. Add vegetables you've eaten throughout the day.
2. The app tracks unique vegetables consumed over a rolling 7-day window.
3. Your progress is shown as a percentage of the 30-vegetable goal.
4. When you're below 100%, the app suggests vegetables you haven't logged recently to help you complete the challenge.

## 🚀 Getting Started

### Prerequisites

- [.NET 9 SDK](https://dotnet.microsoft.com/download) (or later)
- A supported IDE: **Visual Studio 2022+** or **VS Code** with the MAUI extension

### Build & Run

```bash
dotnet restore
dotnet build
dotnet run
```

Or open `VegetableTracker.slnx` in Visual Studio and press **F5**.
